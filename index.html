News

We are happy to release the new 737-700 fuel addon for GeoFS! We hope you guys like the plugin, if there is any bugs report it to this github page.

Fuel System Addon

The 737-700 fuel system is one of the best addons to come to GeoFS, this plugin was made by Yousof, who had the idea to make GeoFS more realistic and to give the user a much better playing experance. If you have any suggestions or you have found something not working, please contact us on github.

How To Use

- Download the javascript file, once done open it and copy the code.
- Open GeoFS then press F12 to open the console.
- Once in the console paste the code you copied from the javascript file, and enjoy!
